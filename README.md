# Trilha Baixo Nível

> Minha trilha de estudos para entender computadores a partir do baixo nível

---

## 📚 Conteúdo Sugerido

- **The C Programming Language** (K&R)
- **Algorithms** (Robert Sedgewick)
- **Computer Organization and Design** (Patterson & Hennessy)
- **Computer Systems: A Programmer's Perspective** (CS:APP)
- **Compilers** ou **Engineering a Compiler**
- **Operating Systems: Three Easy Pieces** (OSTEP)
- **Linux Kernel Development** (Robert Love)

---

## 🎯 Objetivo

Construir uma base sólida sobre como computadores funcionam "por dentro", desde a programação em C até sistemas operacionais e organização de computadores.

---

## 🚀 Como Usar

1. Siga a ordem sugerida dos livros e materiais.
2. Faça anotações e implemente exemplos práticos.
3. Consulte o diretório `material/` para recursos complementares.
